📋 設定步驟

  1. 前往 Google Apps Script
  2. 建立新專案，將 google-apps-script.js 的內容複製到編輯器中
  3. 修改 SHEET_ID 為您的 Google Sheets ID
  4. 部署為 Web 應用程式，設定執行身分為「我」，存取權限為「任何人」
  5. 複製 Web 應用程式 URL 並填入下方設定
  6. 點擊「初始化 Google Sheets」按鈕

🔗 Google Apps Script 設定
